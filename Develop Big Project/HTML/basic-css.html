¿Qué es css3?

Cascade StyleSheets, como su nombre lo indica, es un lenguaje de diseño y estilos visuales de un sitio web cuando este se encuentra compilado en un lenguaje de marcado, como ser HTML. CSS sirve para dar descripción a las formas y la sintaxis del lenguaje de marcado, pudiendo así generar efectos visuales agradables para una mejor experiencia de usuario. Se basa en un modelo de caja común a todos los elementos, pudiéndose manipular cada una de ellas en forma, tamaño, posición, márgenes internos y externos, etc. En la actualidad se utiliza como estándar el CSS3, que a su vez tiene subdivisiones tales como Grid Layout, Flexbox, Bootstrap, Materialize, etc.

Syntax

CSS3 funciona mediante reglas que se aplican a cada etiqueta, clase, ID o conjunto de todas ellas escritas en HTML para otorgarle determinados estilos y posiciones dentro del sitio web. Cada regla lleva la siguiente sintaxis

  • Reglas

    Es la totalidad del texto escrito para un mismo selector

  • Selectores

    Es la referencia hacia la etiqueta, clase o ID escrito en HTML. Algunos de los casos posibles para llamar una etiqueta pueden ser

    • Etiqueta

      Simplemente se escribe a la etiqueta que se quiere modificar. Se suele utilizar en el header o el footer, ya que las demás etiquetas se suelen repetir y no siempre se desea darles el mismo estilo a todas ellas

      header { propiedad: valor; }

    • Etiquetas padre-hijo

      Se utilizan para especificar un único tipo de etiqueta mediante la relación que tiene la misma con sus etiquetas padre. Se utiliza mucho en los elementos de lista y los menús de navegación internos

      nav ul li { propiedad: valor; }

    • Clase

      Además de las etiquetas padre-hijo, para una mayor precisión en la modificación del aspecto de un elemento HTML, se suelen utilizar con frecuencia los selectores de clase. Estos pueden funcionar aislados, es decir, insertando solo el nombre de la clase; o también de manera conjunta con etiquetas del mismo elemento. Su sintaxis se escribe simplemente agregando un punto al principio

      .menu-item { propiedad: valor; } nav ul li.menu-item { propiedad: valor; }

    • ID

      Su uso es muy similar al de las clases, solo que, al ser de uso único de HTML, no se utilizan con tanta frecuencia. Su sintaxis lleva por delante un #

      #list-item { propiedad: valor; }

  • Declaración

    Incluye las propiedades que se le aplican al selector. Puede contener más de una

  • Propiedad

    Indica el atributo a modificar, como color, subrayado, tamaño, etc

  • Valor

    Indica la magnitud de la propiedad. Posee varias unidades de medida
